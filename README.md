# 🛒 Sari-Sari Store

A web-based **Point-of-Sale (POS)** and **Inventory Management System** built for sari-sari stores and small grocery businesses. It helps administrators manage inventory and sales while allowing cashiers to process transactions quickly and efficiently.

## ✨ Features

* 🔐 Role-based authentication (Admin & Cashier)
* 🛍️ Point-of-Sale (POS)
* 📦 Product & inventory management
* 🏷️ Category management
* 💰 Sales & transaction processing
* 📊 Sales reports and analytics
* 🔍 Product search
* 📈 Stock monitoring and updates

## 👥 User Roles

### Admin

* Manage products and categories
* Monitor inventory
* View sales reports
* Manage users
* Track store performance

### Cashier

* Process customer purchases
* Generate receipts
* Search products
* View available stock
